Simplified Supper

Simplified Supper

View The Code

Simplified Supper is intended to make meal planning easy, allowing the user to see at a glance what they have coming up so grocery shopping is a breeze. No longer will users end up throwing away half a bag of carrots because they only used a few of them before they went bad. By looking at the meals the user has planned for the week, Simplified Supper provides suggested recipes based on the percentage of ingredients that can be repurposed from your chosen meals.

Simplified Supper is built in React, styled in Semantic UI for React and utilizes the Yummly API for a portion of the database information.
